## Условные операторы

Условные операторы необходимы для того что бы разделить нашу программу на
некоторые части (ветви). 

В **Python** нет {} - разделяющих блоки кода, для этого
используются отступы для каждого блока. Один отступ это четыре пробела, или tab.

Пример работы:

    if number>100:
        print("Это число больше 100")
    elif namber == 100:
        print("Это число равно 100")
    else:
        print("Это число меньше 100")

Внутрь оператора if может быть вложен другой оператор if, и таких вложений
может быть сколь угодно много.
